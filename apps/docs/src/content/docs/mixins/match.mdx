# @match Mixin

control selector matching like `:not`, `:is`

## Reference

```js title="shilp.config.js"
const shilpConfig = {
	target: "react",

	mixins: {
		match: {
			variants: {
				/* functions */
				not: "&:not(<1>)",
				is: "&:is(<1>)",
				where: "&:where(<1>)",
			},
		},
	},
};

export default shilpConfig;
```
