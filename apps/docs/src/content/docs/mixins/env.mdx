# @env Mixin

environment-based selectors like tall, print

<Alert variant="info">

This page contains only the **mixins configuration reference**. Complete
documentation and usage examples will be added progressively in future updates.

</Alert>

---

<section>

## Reference

```js title="shilp.config.js"
const shilpConfig = {
	target: "react",

	mixins: {
		env: {
			variants: {
				// orientation
				tall: "@media (orientation: portrait)",
				wide: "@media (orientation: landscape)",

				// media type
				print: "@media print",
				screen: "@media screen",
			},
		},
	},
};

export default shilpConfig;
```
