# Text Intent Overview

Text intent contains **element's typography** related utilities.

Utilities available in `@text` intent are listed below.

| Utility      | Description                                                                     | Documentation                                    |
| ------------ | ------------------------------------------------------------------------------- | ------------------------------------------------ |
| `align-*`    | alignment of text (cross and main axis)                                         | [align](/docs/properties/text/align)             |
| `break-*`    | where text breaks                                                               | [break](/docs/properties/text/break)             |
| `case-*`     | text letter casing (uppercase, lowercase, etc.)                                 | [case](/docs/properties/text/case)               |
| `color-*`    | text color                                                                      | [color](/docs/properties/text/color)             |
| `content-*`  | `content` property (`::before` and `::after`)                                   | [content](/docs/properties/text/content)         |
| `family-*`   | font family                                                                     | [family](/docs/properties/text/family)           |
| `gap-*`      | spacing between letters and words                                               | [gap](/docs/properties/text/gap)                 |
| `h-*`        | line height                                                                     | [line height](/docs/properties/text/line-height) |
| `line-*`     | text decoration line                                                            | [line](/docs/properties/text/line)               |
| `lines-*`    | number of visible text lines (line clamping)                                    | [lines](/docs/properties/text/lines)             |
| `move-*`     | length of empty space (indentation) that is put before lines of text in a block | [move](/docs/properties/text/move)               |
| `nums-*`     | numeric glyph styling (tabular, lining, etc.)                                   | [nums](/docs/properties/text/nums)               |
| `overflow-*` | handling of overflowing text                                                    | [overflow](/docs/properties/text/overflow)       |
| `shadow-*`   | text shadow effects                                                             | [shadow](/docs/properties/text/shadow)           |
| `size-*`     | font size                                                                       | [size](/docs/properties/text/size)               |
| `space-*`    | handle white space in text                                                      | [space](/docs/properties/text/space)             |
| `style-*`    | font style (italic, normal, etc.)                                               | [style](/docs/properties/text/style)             |
| `thick-*`    | font thickness (weight)                                                         | [thick](/docs/properties/text/thick)             |
