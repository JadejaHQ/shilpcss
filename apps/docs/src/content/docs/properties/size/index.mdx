# Size Intent Overview

Size intent contains **element's dimentions** related utilities.

Utilities available in `@size` intent are listed below.

| Utility | Description                                                     | Documentation                            |
| ------- | --------------------------------------------------------------- | ---------------------------------------- |
| `h-*`   | Set element's height.                                           | [height](/docs/properties/size/height)   |
| `is-*`  | Set elements size (width & height) including block and inline.  | [is](/docs/properties/size/is)           |
| `max-*` | Set maximum dimensions for elements including block and inline. | [maximum](/docs/properties/size/maximum) |
| `min-*` | Set minimum dimensions for elements including block and inline. | [minimum](/docs/properties/size/minimum) |
| `w-*`   | Set element's width.                                            | [width](/docs/properties/size/width)     |
