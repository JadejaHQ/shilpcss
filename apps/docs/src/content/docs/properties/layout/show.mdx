# show

makes the element visible (use with `hide`)

<Alert variant="info">

This page contains only the **property configuration reference**. Complete
documentation and usage examples will be added progressively in future updates.

</Alert>

---

<section>

## Reference

```js title="shilp.config.js"
const shilpConfig = {
	source: "react",

	properties: {
		layout: {
			show: {
				property: "visibility: <v><i>;",
				values: {
					DEFAULT: "visible",
				},
			},
		},
	},
};

export default shilpConfig;
```
