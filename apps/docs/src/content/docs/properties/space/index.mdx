# Space Intent Overview

Space intent contains **element's spacing** related utilities.

Utilities available in `@space` intent are listed below.

| Utility     | Description                                                  | Documentation                             |
| ----------- | ------------------------------------------------------------ | ----------------------------------------- |
| `gap-*`     | Control spacing between items (flex, grid and multi-column). | [gap](/docs/properties/space/gap)         |
| `margin-*`  | Apply spacing around elements (margin).                      | [margin](/docs/properties/space/margin)   |
| `padding-*` | Apply spacing inside elements (padding).                     | [padding](/docs/properties/space/padding) |
